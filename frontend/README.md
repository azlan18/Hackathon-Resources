# Frontend Resources

## Component Libraries

### React Bits
- **URL**: https://www.reactbits.dev/
- **Description**: 110+ animated React components including text animations, particle backgrounds, and interactive effects
- **Variants**: JS+CSS, JS+Tailwind, TS+CSS, TS+Tailwind
- **License**: MIT (Free for commercial use)

### Aceternity UI
- **URL**: https://ui.aceternity.com/
- **Description**: 200+ free copy-paste components built with Tailwind CSS and Framer Motion
- **Features**: Animated modals, expandable cards, 3D effects, smooth transitions
